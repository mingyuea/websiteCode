Tic Tac Toe

This is a Tic Tac Toe program written in Javascript. It is written to never lose to the player(or at least, that was the original intent). I had originally set out with the ambitious intent to write a basic AI of sorts, but quickly found myself out of my league. I instead fell to utilizing a magic square, something I had learned from Matlab programming. A magic square was a square composed of numbers that when added together horizontally, vertically, or diagonally in any direction, would sum to the same number. The program used a series of sum checks and if/else statements to determine the best possible move. Later, I had read about something called Minimax recursion that is typically used in these types of decision making games, which is something quite interesting that I might explore. If this script could even be considered an AI at all, it would probably be a rather dim one.

Color Changing Grids

This was just a private little project, one of my first forays into Javascript coding. It is very ugly, and lacks in any sort of decoration. The exercise was done to help better my understanding of Javascript and HTML DOM. It was a problem posed to me by a friend: "Create a 4x4 colored grid, every 250 milliseconds have one grid randomly change color. Once it has changed color, that grid shall not change color for at least another 2 seconds". JQuery or CSS tricks were not allowed, only plain ol' Javascript.

Close-Knit Words

This was a challenge I saw on the Code Golf StackExchange site, that I decided to try. Code Golf, for those that are unfamiliar, is to solve a puzzle or a problem following golf rules(and thus using the least lines or bytes of code as possible). Solving puzzles on Code Golf is a fun hobby I do, though I'm not nearly skilled enough to golf all my code just yet. It's always mind-blowing to see some of the pros golf their codes in as less bytes as possible. I mainly used this challenge to practice nested loops (something I still felt fairly weak at) and insertion sort.

Fading an Image, Left to Right

JQuery is amazing for certain animations to HTML elements. It of course includes its a built in fade function. However, a friend asked me if it was possible to fade an image from left to right. JQuery was somewhat lacking, so I decided to build my own fading function. It was written primarily in Javascript. The effect was done by setting the image as a background-image of a table element, then slowly changing the opacity of the table (and thus the image as well) column by column. Feel free to use it or change it, as you like.
